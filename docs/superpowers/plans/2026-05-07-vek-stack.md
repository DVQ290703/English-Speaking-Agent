# VEK Stack — Elasticsearch Data Streams Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Migrate the VEK log pipeline from a single daily index to three routed Elasticsearch data streams, with automated bootstrap on `docker compose up` and Kibana data views ready for investigation.

**Architecture:** Fix `AuditLogger` to emit pure JSON, update Vector to route by log category, provision Elasticsearch via an init container (ILM policy + component template + index template), and register Kibana data views via a second init container.

**Tech Stack:** Python (AuditLogger fix), Vector VRL (route transform), Elasticsearch 8.13 REST API, Kibana 8.13 Saved Objects API, Docker Compose, curlimages/curl:8

## Task 2: Update Vector config — route transform + data stream sinks

**Files:**
- Modify: `deployments/vector/vector.yaml`

- [ ] **Step 1: Replace `deployments/vector/vector.yaml` entirely**

```yaml
# Vector configuration — tails app log files, parses JSON spans, routes to data streams

sources:
  app_logs:
    type: file
    include:
      - /logs/*.log
    read_from: beginning

transforms:
  parse_json_logs:
    type: remap
    inputs:
      - app_logs
    source: |
      # Log line format: "2026-05-07 10:23:45 INFO     [AI-Lab-Agent]: <message>"
      parsed, err = parse_regex(.message, r'^\S+ \S+ \S+\s+\[\S+\]:\s+(?P<payload>.+)$')
      if err == null {
        json_payload, json_err = parse_json(parsed.payload)
        if json_err == null && is_object(json_payload) {
          . = merge!(., json_payload)
          .log_type = "span"
        } else {
          .log_type = "plain"
          .raw_message = parsed.payload
        }
      } else {
        .log_type = "plain"
        .raw_message = .message
      }
      .source_file = .file

  route_by_category:
    type: route
    inputs:
      - parse_json_logs
    route:
      audit: exists(.event_type)
      span: .log_type == "span" && !exists(.event_type)
      plain: .log_type == "plain"

sinks:
  elasticsearch_audit:
    type: elasticsearch
    inputs:
      - route_by_category.audit
    endpoints:
      - http://elasticsearch:9200
    mode: data_stream
    data_stream:
      type: logs
      dataset: voice_agent.audit
      namespace: default
    encoding:
      except_fields:
        - file
        - host
        - source_type

  elasticsearch_spans:
    type: elasticsearch
    inputs:
      - route_by_category.span
    endpoints:
      - http://elasticsearch:9200
    mode: data_stream
    data_stream:
      type: logs
      dataset: voice_agent.spans
      namespace: default
    encoding:
      except_fields:
        - file
        - host
        - source_type

  elasticsearch_plain:
    type: elasticsearch
    inputs:
      - route_by_category.plain
    endpoints:
      - http://elasticsearch:9200
    mode: data_stream
    data_stream:
      type: logs
      dataset: voice_agent.plain
      namespace: default
    encoding:
      except_fields:
        - file
        - host
        - source_type
```

  elasticsearch-init:
    image: curlimages/curl:8
    container_name: voice_agent_elasticsearch_init
    depends_on:
      elasticsearch:
        condition: service_healthy
    restart: "no"
    command: >
      sh -c "
        echo 'Creating ILM policy...' &&
        curl -f -s -X PUT http://elasticsearch:9200/_ilm/policy/voice-agent-logs-policy
          -H 'Content-Type: application/json'
          -d '{\"policy\":{\"phases\":{\"hot\":{\"actions\":{\"rollover\":{\"max_primary_shard_size\":\"10gb\",\"max_age\":\"7d\"}}},\"delete\":{\"min_age\":\"30d\",\"actions\":{\"delete\":{}}}}}}'
        &&
        echo 'Creating component template...' &&
        curl -f -s -X PUT http://elasticsearch:9200/_component_template/voice-agent-logs-mappings
          -H 'Content-Type: application/json'
          -d '{\"template\":{\"mappings\":{\"dynamic\":\"false\",\"properties\":{\"@timestamp\":{\"type\":\"date\"},\"log_type\":{\"type\":\"keyword\"},\"trace_id\":{\"type\":\"keyword\"},\"session_id\":{\"type\":\"keyword\"},\"user_id\":{\"type\":\"keyword\"},\"span_name\":{\"type\":\"keyword\"},\"span_kind\":{\"type\":\"keyword\"},\"duration_ms\":{\"type\":\"float\"},\"status\":{\"type\":\"keyword\"},\"event_type\":{\"type\":\"keyword\"},\"raw_message\":{\"type\":\"text\"},\"source_file\":{\"type\":\"keyword\"},\"environment\":{\"type\":\"keyword\"},\"model\":{\"type\":\"keyword\"},\"prompt_tokens\":{\"type\":\"integer\"},\"completion_tokens\":{\"type\":\"integer\"},\"total_tokens\":{\"type\":\"integer\"},\"estimated_cost_usd\":{\"type\":\"float\"}}}}}'
        &&
        echo 'Creating index template...' &&
        curl -f -s -X PUT http://elasticsearch:9200/_index_template/voice-agent-logs
          -H 'Content-Type: application/json'
          -d '{\"index_patterns\":[\"logs-voice_agent.*-*\"],\"data_stream\":{},\"composed_of\":[\"voice-agent-logs-mappings\"],\"template\":{\"settings\":{\"index.lifecycle.name\":\"voice-agent-logs-policy\",\"number_of_shards\":1,\"number_of_replicas\":0}},\"priority\":200}'
        &&
        echo 'Elasticsearch bootstrap complete.'
      "
